# Calculator
A calculator project using html, css and js
with buttons for each digit and operator including = and clear button
✅ Does basic math (add, subtract, multiply, divide)
✅ Has a display and buttons (0-9, operators, =, clear)
✅ Chains operations (12 + 7 - 1 = 18)
✅ Handles edge cases (÷0, decimals, consecutive operators)
✅ Extra Credit: Decimal point, backspace, keyboard support
